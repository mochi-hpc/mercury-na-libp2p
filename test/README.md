# Test files

Most of the files in this directory have been copied or adapted from the
[Mercury](https://github.com/mercury-hpc/mercury) test suite
(`Testing/na/` and `Testing/mercury/` in the Mercury source tree).

They are provided here so that an out-of-tree NA plugin can be tested
without requiring access to the full Mercury source.
